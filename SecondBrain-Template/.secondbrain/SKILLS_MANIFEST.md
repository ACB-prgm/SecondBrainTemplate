# Skills Manifest

Optional skills extend SecondBrain for less-common workflows. Read a referenced skill file only when the user asks for that workflow or the current task clearly requires it.

| Skill | Purpose | Reference |
|---|---|---|
| Onboarding | Guide first-run setup or explicit onboarding by getting the user to a useful first result quickly without a long upfront tutorial. | `.skills/ONBOARDING.md` |
| Bug Reporting | Report SecondBrain template, prompt, schema, action, workflow, or integration issues in the template/dev repo. | `.skills/ISSUE_CREATION.md` |
| Tabular Storage | Store scalable row/column datasets such as expenses, transactions, budgets, inventory, logs, recurring measurements, or other growing row-based datasets in Google Sheets while keeping GitHub markdown as the source-of-truth context layer. | `.skills/TABULAR_STORAGE.md` |
